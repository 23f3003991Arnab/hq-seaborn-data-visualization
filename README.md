# Customer Satisfaction Analytics - Seaborn Visualization

## Project Overview
This project creates a professional-grade Seaborn visualization analyzing customer satisfaction scores across different product categories for Douglas Buckridge's retail client.

## Contact Information

## Project Structure
```
├── README.md          # This file
├── chart.py          # Python script for visualization
└── chart.png         # Generated chart (512x512 pixels)
```

## Requirements
- Python 3.x
- seaborn
- matplotlib
- pandas

## Installation
```bash
pip install seaborn matplotlib pandas
```

## Usage
Run the script to generate the visualization:
```bash
python chart.py
```

This will create `chart.png` with exactly 512x512 pixel dimensions.

## Visualization Details
- **Chart Type**: Horizontal bar plot
- **Data**: Customer satisfaction scores (1-5 scale) across 8 product categories
- **Styling**: Professional whitegrid theme with Blues color palette
- **Output**: 512x512 pixel PNG image

## Business Context
The visualization provides actionable insights for executive decision-making, showing comparative customer satisfaction across product categories to inform strategic planning and resource allocation.
